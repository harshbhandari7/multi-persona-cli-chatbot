# Multi-Persona CLI Chatbot

A CLI chatbot that supports multiple personas, prompt strategies, and model benchmarking.

## Features

- Persona switching via YAML
- Prompt comparison (zero-shot, few-shot, chain-of-thought)
- Multi-model benchmarking
- Token usage tracking
- Cost estimation
- Conversation logging

## CLI Usage

```python main.py --persona tech_support```

```python main.py --compare```

```python main.py --benchmark```
